SpaceAndroid
============

This is an Android implementation of Space - end-to-end encrypted, blockchain-backed, data storage.

Setup
=====
Libraries

    mkdir app/libs
    ln -s <bcandroidaar> bc-android/app-debug.aar
    ln -s <spacejavalib> app/libs/SpaceJava.jar

Build
=====

    ./gradlew build
